# Salesforce Support Center – Pagination Module

## Overview
This project improves the usability of a Salesforce Support Center by adding pagination
and page-size control to ticket listings. It is designed to handle large datasets smoothly
while keeping the UI responsive.

## What This Project Solves
- Slow loading of large ticket lists
- Poor user navigation experience
- Unnecessary database load

## Key Features
- Pagination with page navigation
- Configurable page size
- Optimized Apex queries
- Clean and simple UI logic

## Architecture Overview
User selects page →  
Lightning component requests data →  
Apex controller returns limited records →  
UI updates dynamically

## Technologies Used
- Lightning Aura Components
- Apex Controllers
- SOQL
